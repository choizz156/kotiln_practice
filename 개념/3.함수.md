## 함수 표현
```
- 블록 본문
fun 함수이름(p1: 타입1, p2: 타입2, ...)(: 반환타입){

  return ...
}

- 식 본문
fun 함수이름(p1: 타입1, p2: 타입2, ...)(: 반환타입) = 수식
```

```kotlin
fun multiplyByTwo(x: Int): Int {
    println("2곱함")
    return x*2
}

fun hello(): Unit{ //Unit == void 생략가능
    println("hello")
}
fun helloY(){
    println("helloY")
}

fun multiplyBy3(x: Int): Int = x * 3
fun multiplyBy4(x: Int) = x * 4  //반환 타입 생략 가능
```
