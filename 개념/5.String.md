### raw string `"""`
- 이스케이프 문자 없이 문자열 작성 가능
- 여러 줄에 걸쳐 작성 가능 (\n 같은 특수문자 사용 없이 줄바꿈 가능)
- 일반 문자열보다 가독성이 높음
-  코드 들여쓰기 때문에 출력 시 공백이 포함됨

#### trimMargin
- trimMargin() 함수는 각 줄 앞에 지정한 기호까지의 공백을 제거함
- 기본 기호는 | (파이프 문자)이며, 다른 기호도 사용 가능
```kotlin
 val nurseryRhyme = """
   *Humpty Dumpty sat on a wall
    |Humpty Dumpty had a great fall
    |All the king's horses and all the king's men
    |Couldn't put Humpty together again
""".trimMargin("*")
println(nurseryRhyme)

//결과
Humpty Dumpty sat on a wall
    |Humpty Dumpty had a great fall
    |All the king's horses and all the king's men
    |Couldn't put Humpty together again
```

- `$`를 사용해서 변수를 대입할 수 있음.
- `${}` {}안에 수식이나 조건식을 넣을 수 있음. 
- `"""`안에도 전부 가능함.
- 문자열 안에 변수명이 아닌 문자로써 $를 출력하려면 $를 이스케이프해야 함
  - 변수명이 아닌 경우엔 그냥 $만 써도 됨
- 외부 객체의 프로퍼티를 문자열 안에서 접근하려면 중괄호 사용 필수
```kotlin
fun main() {
    val answer = 42
    
    
    println("found $answer!")
    println("found $123!")


    val cond = true;
    println(
        "${if(cond) 'a' else 'b'}"
    )

    val x = 10
    println("$x + 4 = ${x + 4}")

    val s = "key"

    println(""" s = "$s" """)
    println(""" s = "${x + 3}" """)

    println("Employee ID: ${employee.id}") 
}
```
